## 1. 使用说明

- 本矩阵结构固定为：`FR -> Backlog -> 验收条目 -> 自动化用例/验证脚本 -> 证据产物`。
- P0 条目必须五段全量映射，不允许留空。
- 证据产物统一归档到仓库内路径，禁止使用外部系统链接作为唯一证据。

## 3. 冻结条件

追踪矩阵进入冻结态需满足：

- P0 行全部具备可执行的自动化入口或验证脚本。
- 验收清单 P0 项全部打勾并附证据路径。
- Backlog 中对应 P0 项全部满足关闭标准。

## 4. 维护规则

- 新增或修改 FR 时，必须同步修改本矩阵。
- Backlog ID、验收 ID 变更后，24 小时内完成矩阵回填。
- 每次里程碑评审前由 QA Lead 检查证据路径可访问性。
