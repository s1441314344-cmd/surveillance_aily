# 摄像头中心索引与共享上下文

## 1. 文档定位
本文件同时承担“模块索引”和“共享上下文”两种职责，用于统一描述摄像头中心的多子路由结构和共享状态。

## 2. 页面清单
| 路由 | 页面 | 说明 |
|------|------|------|
| `/cameras/devices` | 设备管理 | 维护摄像头基础信息、拍照和录像入口 |
| `/cameras/monitoring` | 监测配置 | 维护触发规则、信号监测配置和实时调试 |
| `/cameras/media` | 媒体中心 | 查看抓拍和录像产物 |
| `/cameras/diagnostics` | 诊断中心 | 查看状态检查、深度诊断和状态日志 |

## 3. 共享状态
- `selectedCameraId`
  - 当前在摄像头中心被选中的摄像头
  - 通过查询参数与页面状态同步
- `cameraSearch`
  - 左侧列表搜索条件
- `alertOnly`
  - 左侧列表是否只展示异常设备
- `cameraStatusMap`
  - 当前设备状态摘要

## 4. 共享规则
- 摄像头左侧列表与当前子页共享同一套选中设备上下文。
- 子页切换不应丢失当前选中摄像头。
- 所有子页都依赖 `Camera`、`CameraStatusLog` 和摄像头状态摘要，但各自只定义本页私有行为。

## 5. 阅读顺序
1. 先读本文件
2. 再读对应子页 PRD
3. 若涉及任务联动，补读 `jobs-center.md`
